ส่งคำร้องเรียนสำเร็จ!

ระบบได้รับเรื่องของคุณแล้ว กรุณาเก็บรหัส Ticket ID นี้เพื่อใช้ติดตามความเคลื่อนไหว

รหัสติดตามเรื่องของคุณ

สภานักเรียนจะตรวจสอบและอัปเดตสถานะให้เร็วที่สุด
นโยบายความเป็นส่วนตัว
1. การเก็บรวบรวมข้อมูล

สภานักเรียนจะเก็บรวบรวมข้อมูลส่วนบุคคลที่ท่านระบุในฟอร์ม เช่น ชื่อ-นามสกุล อีเมล และข้อมูลการร้องเรียน เพื่อใช้ในการประสานงานติดตามเรื่องเท่านั้น

2. วัตถุประสงค์

เพื่อนำข้อมูลไปตรวจสอบข้อเท็จจริง และดำเนินการแก้ไขปัญหาตามกระบวนการของโรงเรียนสุรวิทยาคาร

3. การคุ้มครองข้อมูล

ข้อมูลของท่านจะถูกเก็บรักษาเป็นความลับ และจะเข้าถึงได้เฉพาะผู้มีส่วนเกี่ยวข้องในกระบวนการพิจารณาเรื่องร้องเรียนเท่านั้น

4. ระยะเวลาการเก็บข้อมูล

เราจะเก็บข้อมูลไว้จนกว่ากระบวนการดำเนินการจะเสร็จสิ้น และจะจัดเก็บไว้ในฐานข้อมูลสถิติของสภานักเรียน

หากพบปัญหากรุณาติดต่อสภานักเรียน หรือผู้พัฒนาเว็บไซต์

คำถามที่พบบ่อย (FAQ)

ปลอดภัยแน่นอนครับ! ระบบจะทำการซ่อนข้อมูลผู้แจ้งออกจากฐานข้อมูลหลัก แม้แต่แอดมินหรือสภานักเรียนก็ไม่สามารถดูชื่อ-นามสกุลของคุณได้หากคุณเลือกตัวเลือก "ไม่ระบุตัวตน"

โดยปกติสภานักเรียนจะเข้ามารับเรื่องภายใน 24-48 ชั่วโมง และระยะเวลาการดำเนินการจะขึ้นอยู่กับความยากง่ายของปัญหา หากเป็นเรื่องเร่งด่วนทีมงานจะรีบประสานงานฝ่ายที่เกี่ยวข้องทันทีครับ

เพื่อนๆ สามารถแนบรูปภาพหลักฐานได้สูงสุด **3 รูป** โดยรวมขนาดไม่เกิน 5MB เพื่อให้แอดมินสามารถเห็นภาพปัญหาที่เกิดขึ้นได้อย่างชัดเจนที่สุด

ทุกเรื่องที่เป็นประโยชน์ต่อส่วนรวมครับ เช่น อาคารสถานที่ชำรุด, ข้อเสนอแนะเรื่องกิจกรรม, ปัญหาเรื่องสวัสดิการ หรือการปรับปรุงระบบการเรียนการสอน (เรื่องส่วนตัวหรือการกลั่นแกล้งกันอาจมีการพิจารณาเป็นรายกรณี)

หากระบุอีเมลไว้ ระบบจะส่ง Ticket ID ไปให้ทางอีเมลครับ แต่ถ้าแจ้งแบบไม่ระบุตัวตนและลืมรหัส แนะนำให้แจ้งเรื่องเข้ามาใหม่อีกครั้งพร้อมระบุว่าเป็นเรื่องเดิมที่แจ้งไว้ครับ
